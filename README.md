Compile with ```make``` and run ```playlistdriver```.

# Commands

1. ```m <size>```
   - Creates playlist of specified size.
2. ```i <song>;<artist>```
   - Adds song to playlist.
3. ```p <position>```
   - Plays song at position.
4. ```e <position>```
   - Erases song at position.
5. ```exit```
   - Closes program.
